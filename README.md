## 🤖 AI-augmented engineering

The part I'm most excited about right now: making AI coding assistants genuinely useful on a complex, proprietary enterprise platform — not just autocomplete, but reliable, schema-aware help.

- **Custom MCP server** — designed and built a [Model Context Protocol](https://modelcontextprotocol.io) server in **Python** that exposes Maximo metadata, schema, and database tooling to AI assistants. This turns vague, documentation-based guesswork into schema-validated answers and cuts root-cause analysis on tricky configuration issues from hours to minutes.
- **Empirical API registry** — a subsystem that probes and records the platform's *actual* Java API surface (business-object classes and methods), so generated and hand-written scripts can be validated against real method signatures instead of assumptions — which sharply reduces deployment failures.
- **Automated auditing / linting** — a custom rule set wired into the dev workflow to enforce coding standards and catch antipatterns *before* deployment.
- **Tooling-in-the-loop development** — daily use of agentic assistants (Claude, Cline) integrated with the tooling above to accelerate analysis, code generation, and documentation.
